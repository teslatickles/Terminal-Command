This Node script accepts flags in terminal to perform various os-related functions (mkdir, ls, touch). 

To make a directory:

`node index.js _-mkdir_ nihongo`

To create a file:

`node index.js _-touch_ bakka.txt`

To list the contents of a directory:

`node index.js -ls`
